Memory Card Game

A emoji card memory game. Built on a LAMP stack which includes features such as: login, leaderboard, register. Each level differs in number of cards present on board, more than 15 attempts or 10minutes and you lose.

SecureNotes

Built with Django, SecureNotes ensures the security and privacy of user notes through encryption. Each note sent to the backend is encrypted using Fernet (symmetric encryption) before being saved to the database. When a user requests to view their notes, the data is decrypted.

Classic Snake Game

Snake game built using JavaScript, HTML, and CSS. The objective is to control the snake, guiding it to eat fruits. Each time the snake eats a fruit, it grows longer.if the snake touches the borders of the game area or if its head collides with its body, the game is over.
